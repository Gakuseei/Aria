Aria One-Click Tools
====================

This folder contains automated installers for Aria's features.
Simply double-click the script you need.

[1] Core System
- install_ollama.bat  -> Installs the AI Brain (Required)

[2] Voice Features
- install_piper.bat   -> Installs Standard Voice (Fast/Light)
- install_zonos.bat   -> Installs Premium Voice (High Quality, requires 6GB VRAM)
- start_zonos.bat     -> STARTS the Premium Voice engine

[3] Image Features
- install_stability.bat -> Installs Stability Matrix (for Image Generation)
  (Use this to install 'Stable Diffusion WebUI' or 'Forge')

Troubleshooting:
- Ensure you have an internet connection.
- If a script closes immediately, try running it as Administrator.
